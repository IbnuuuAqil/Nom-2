Fungsi Program
Program ini dibuat untuk membantu kita mengetahui apakah angka yang dimasukkan pengguna termasuk bilangan ganjil atau genap. Dengan program ini, kita tidak perlu lagi mengecek angka secara manualâ€”tinggal masukkan angkanya, lalu program akan langsung memberikan jawabannya.

Cara Kerja Program
Cara kerja program ini cukup sederhana. Pertama, program akan meminta pengguna memasukkan sebuah angka. Setelah angka dimasukkan, program akan melakukan pengecekan dengan menggunakan operasi modulus (%), yaitu memeriksa sisa hasil pembagian angka tersebut dengan angka 2. Jika sisanya 0, berarti angka itu adalah bilangan genap. Jika sisanya bukan 0, maka angka tersebut termasuk bilangan ganjil. Setelah pengecekan selesai, program akan menampilkan hasilnya kepada pengguna sehingga pengguna langsung tahu apakah angka yang dimasukkan ganjil atau genap.
